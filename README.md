# Student Management System (C++)

A console-based C++ application to manage student records.

## Features
- Add, display, search, update, delete students
- Auto grade calculation based on marks
- Data saved to file (persists after program closes)
- Class statistics (average, highest, lowest marks)

## Concepts Used
- Object-Oriented Programming (Classes & Objects)
- File I/O (fstream)
- STL Vectors
- Loops & Iterators
- Input validation & error handling

## How to Run
```bash
g++ student_management.cpp -o sms
./sms
```

## Tech Stack
- Language: C++
- Compiler: g++
